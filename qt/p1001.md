# 欢迎使用 Markdown 编辑器

**这是一个基于 Editor.md 的 Markdown 编辑器**

您可以在此编辑 Markdown 内容，并将其保存到 GitHub 仓库中。支持中文、数学公式、流程图等多种格式。

## 示例内容

### 数学公式
$E=mc^2$

### 流程图
```flow
st=>start: 开始
e=>end: 结束
op1=>operation: 编辑 Markdown
op2=>operation: 设置文件路径
op3=>operation: 提交到 GitHub
cond=>condition: 操作成功?

st->op1->op2->op3->cond
cond(yes)->e
cond(no)->op1
```
